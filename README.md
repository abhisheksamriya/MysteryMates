# Real-time, Room-Based Anonymous Chat App (WebSockets, React, Node.js)

MysteryMates is a real-time private chat platform where users can instantly join a room and start chatting anonymously.

## Demo

https://mystery-mates.vercel.app/

## Features

- No sign up
- Instant room join
- Anonymous usernames
- Room-Based Communication
- Real-Time Messaging
- Auto-Disconnect on Exit

## Tech Stack

**Client:** React + TypeScript, TailwindCSS

**Server:** Node, WebSocket, TypeScript

## Running Tests

To run backend

```bash
  cd server
  npm install
  npm run dev
```

To run frontend

```bash
  cd client
  npm install
  npm run dev
```
